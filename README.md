# Конструктор боя для гитары

Интерактивный инструмент для создания и визуализации гитарных боёв с синхронизацией звука и визуальных подсказок.

## Установка

```bash
npm install
```

## Запуск без сервера

После установки зависимостей, приложение можно запустить без сервера:

1. Соберите проект:
```bash
npm run build
```

2. Откройте файл `dist/index.html` в браузере напрямую (через file:// протокол)

## Разработка

Для разработки используйте:
```bash
npm run dev
```

## Структура проекта

- `js/` - Исходный код JavaScript
- `css/` - Стили Tailwind CSS
- `dist/` - Собранный проект для запуска без сервера
- `webpack.config.js` - Конфигурация Webpack

## Особенности

- Поддержка различных темпов (4, 8, 16 долей)
- Синхронизация визуальных подсветок с аудио
- Экспорт в PNG и JSON
- Поддержка аккордовых прогрессий
